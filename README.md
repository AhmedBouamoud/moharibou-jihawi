# محاربي الجهوي

منصة مراجعة دينامية لمادة الاجتماعيات — السنة الثالثة إعدادي.
إعداد: ذ. أحمد بوعمود — مؤسسة الحنان الخاصة | 2025-2026

## المزايا
- 18 درساً (تاريخ، جغرافيا، تربية على المواطنة).
- بنك أسئلة موسّع: 125 سؤالاً.
- معركة المعرفة + محاكي امتحان (اختيار المادة وعدد الأسئلة).
- ركن الأستاذ: تعليق تشجيعي + نصائح وحيل في نهاية كل مرحلة.
- الباحث الذكي: بحث داخلي + مساعد ذكي تفاعلي (عند الاتصال).
- بطاقات سحرية، مهام وأوسمة، شهادة تقدير قابلة للطباعة.
- تطبيق PWA قابل للتثبيت ويعمل دون اتصال (service worker).
- صفر تبعيات خارجية.
